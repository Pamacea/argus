# /argus-learn

Manually teach ARGUS something important.

## Usage

```
/argus-learn <type>: <content>
```

## Examples

```
/argus-learn pattern: Always use JWT tokens for API authentication
/argus-learn command: npm run test:unit runs only unit tests
/argus-learn decision: We chose PostgreSQL over MongoDB for relational data
```

## Types

- `pattern` - Code patterns and conventions
- `command` - CLI commands and workflows
- `decision` - Technical decisions with rationale
- `solution` - Working solutions to problems
- `best-practice` - Best practices and guidelines
