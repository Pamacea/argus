# /argus-search

Search your ARGUS memory for relevant context.

## Usage

```
/argus-search <query>
```

## Examples

```
/argus-search How did we fix the authentication bug?
/argus-search What patterns do we use for API calls?
/argus-search Where is the user service located?
```

## What it does

- Performs semantic search across all stored memories
- Returns relevant past solutions, patterns, and decisions
- Includes context like file locations and timestamps
- Shows similarity scores for each result
