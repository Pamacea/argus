# ARGUS v0.8.0 - Rust Rewrite CLI-First

> **Breaking Change:** Abandon du MCP, CLI natif avec intégration via Claude Code Rules

---

## 🎯 Vision v0.8.0

ARGUS devient un **CLI Rust autonome** qui améliore Claude Code via :

1. **Installation globale** - `cargo install argus-rs`
2. **Rules injection** - `argus init` ajoute `argus.md` à `~/.claude/rules/`
3. **Memory engine** - SQLite local pour transactions et indexation
4. **CLI commands** - Interface directe, pas de MCP

### Inspiration Directe : RTK.md

RTK s'installe et ajoute ses règles au global Claude Code. ARGUS fait la même chose :

```
~/.claude/rules/argus.md  # Injecté par `argus init`
```

## 🚀 Commandes CLI v0.8.0

```bash
# Installation
cargo install argus-rs

# Initialisation (crée ~/.argus/ + injecte rules)
argus init

# Gestion mémoire
argus remember "Added JWT authentication to user service"
argus recall "authentication patterns"
argus recall --limit 5 "git hooks"

# Indexation
argus index                    # Indexe le projet courant
argus index --watch            # Mode watch
argus index --stats            # Statistiques

# Inspection
argus list                     # Liste les transactions
argus show <id>                # Montre une transaction
argus stats                    # Statistiques globales

# Configuration
argus config get <key>
argus config set <key> <value>

# Nettoyage
argus prune --before <date>    # Supprime anciennes transactions
argus reset                    # Reset complet (danger!)
```

## 📊 Schema SQLite

```sql
-- transactions.sql
CREATE TABLE transactions (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    prompt TEXT NOT NULL,
    prompt_type TEXT NOT NULL,  -- 'user'|'tool'|'system'
    summary TEXT,
    intent TEXT,
    context TEXT,               -- JSON
    result TEXT,                -- JSON
    metadata TEXT,              -- JSON
    created_at INTEGER NOT NULL,
    session_id TEXT,
    project_path TEXT
);

CREATE INDEX idx_transactions_created ON transactions(created_at);
CREATE INDEX idx_transactions_session ON transactions(session_id);
CREATE INDEX idx_transactions_project ON transactions(project_path);

-- search index (FTS5)
CREATE VIRTUAL TABLE transactions_fts USING fts5(
    prompt, summary, intent,
    content=transactions,
    content_rowid=id
);

-- projects.sql
CREATE TABLE projects (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    path TEXT UNIQUE NOT NULL,
    name TEXT,
    indexed_at INTEGER,
    file_count INTEGER,
    total_size INTEGER
);

-- indexed_files.sql
CREATE TABLE indexed_files (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    project_id INTEGER NOT NULL,
    path TEXT NOT NULL,
    language TEXT,
    hash TEXT,
    indexed_at INTEGER,
    FOREIGN KEY (project_id) REFERENCES projects(id)
);
```

## 🚀 Quick Start Implementation

### Semaine 1: Core + CLI de base

```bash
cargo new argus-rs --bin
cd argus-rs
cargo add clap tokio rusqlite serde_json
```

```rust
// crates/cli/src/main.rs
use clap::{Parser, Subcommand};
use anyhow::Result;

#[derive(Parser)]
#[command(name = "argus")]
#[command(version = "0.8.0")]
struct Cli {
    #[command(subcommand)]
    command: Commands,
}

#[derive(Subcommand)]
enum Commands {
    /// Initialize ARGUS (create ~/.argus/ + inject rules)
    Init,
    /// Remember a transaction
    Remember { description: String },
    /// Recall past transactions
    Recall { query: String, #[arg(short, default = "10")] limit: usize },
    /// Index current project
    Index,
    /// Show statistics
    Stats,
}

#[tokio::main]
async fn main() -> Result<()> {
    let cli = Cli::parse();
    match cli.command {
        Commands::Init => cmd_init().await?,
        Commands::Remember { description } => cmd_remember(description).await?,
        Commands::Recall { query, limit } => cmd_recall(query, limit).await?,
        Commands::Index => cmd_index().await?,
        Commands::Stats => cmd_stats().await?,
    }
    Ok(())
}
```
